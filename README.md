# Hockey's Club Website

A modern, responsive website for a professional hockey club.

## Features

- **Responsive Navigation Menu** with call-to-action
- **Hero Section** with slider and bold imagery
- Key club stats (e.g., Player Facility, Experienced Coach, etc.)
- Sections for different hockey programs: Junior, Teenager, Professional
- **Featured Products** grid with price and status tags
- Accordion for frequently asked client question

## Technologies Used

- **HTML5**  
- **CSS3**
- **Tailwind CSS** – For rapid, utility-first UI styling  
- **JavaScript** – For interactive elements like sliders and accordions

## Live Preview

## How to Use

1. Clone this repository or download the ZIP.
2. Make sure [Tailwind CSS](https://tailwindcss.com/docs/installation) is included (via CDN recommended for simple projects).
3. Customize content in `index.html` as needed.
4. Edit or add CSS in `style.css` for any extra custom styles (optional).
5. Update images and assets in `/assets/images`.
6. Modify/extend interactivity in `/js/main.js`.

## Customization

- **Brand and Program Details**: Update the text, images, and links in the HTML to fit your club.
- **Styling**: Tailwind utility classes can be changed for color, spacing, typography, etc. as desired.
- **Products**: Add or remove product cards as needed.
- **Contact Info**: Update with correct phone, email, and address.
